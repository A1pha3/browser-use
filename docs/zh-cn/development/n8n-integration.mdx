## 概述

Browser Use 可以通过社区节点与 [n8n](https://n8n.io)（工作流自动化平台）集成。此集成允许你直接从 n8n 工作流触发浏览器自动化任务。

## 安装 n8n 社区节点

有几种方法可以在 n8n 中安装 Browser Use 社区节点：

### 使用 n8n Desktop 或 Cloud

1. 导航到 **设置 > 社区节点**
2. 点击 **安装**
3. 在 **名称** 字段中输入 `n8n-nodes-browser-use`
4. 点击 **安装**

### 使用自托管 n8n 实例

在你的 n8n 安装目录中运行以下命令：

```bash
npm install n8n-nodes-browser-use
```

### 开发使用

如果你想使用 n8n 节点进行开发：

1. 克隆仓库：
   ```bash
   git clone https://github.com/draphonix/n8n-nodes-browser-use.git
   ```
2. 安装依赖：
   ```bash
   cd n8n-nodes-browser-use
   npm install
   ```
3. 构建代码：
   ```bash
   npm run build
   ```
4. 链接到你的 n8n 安装：
   ```bash
   npm link
   ```
5. 在你的 n8n 安装目录中：
   ```bash
   npm link n8n-nodes-browser-use
   ```

## 设置 Browser Use Cloud API 凭据

要在 n8n 中使用 Browser Use 节点，需要配置 API 凭据：

1. 在 [Browser Use Cloud](https://cloud.browser-use.com/new-api-key) 注册账号
2. 导航到设置或 API 部分
3. 生成或复制你的 API Key
4. 在 n8n 中创建新的凭据：
   - 转到 **凭据** 选项卡
   - 点击 **创建新的**
   - 选择 **Browser Use Cloud API**
   - 输入你的 API Key
   - 保存凭据

## 使用 Browser Use 节点

安装完成后，你可以将 Browser Use 节点添加到你的工作流中：

1. 在工作流编辑器中，在节点面板中搜索"Browser Use"
2. 将节点添加到你的工作流
3. 设置凭据
4. 选择保存的凭据
5. 选择操作：
   - **运行任务**：使用自然语言指令执行浏览器自动化任务
   - **获取任务**：检索任务详情
   - **获取任务状态**：检查任务执行状态
   - **暂停/恢复/停止任务**：控制运行中的任务
   - **获取任务媒体**：获取截图、视频或 PDF
   - **列出任务**：获取任务列表

### 示例：运行浏览器任务

以下是如何使用 Browser Use 节点运行浏览器任务的简单示例：

1. 将 Browser Use 节点添加到你的工作流
2. 选择"运行任务"操作
3. 在"指令"字段中，输入你对浏览器要做什么的自然语言描述，例如：
   ```
   访问 example.com，截取首页截图，提取所有主要标题文本
   ```
4. 可选启用"保存浏览器数据"以保留 Cookie 和会话信息
5. 将节点连接到后续节点以处理结果

## 工作流示例

Browser Use n8n 节点支持各种自动化场景：

- **网页抓取**：按计划从网站提取数据
- **表单填写**：跨 Web 应用程序自动执行数据输入
- **监控**：检查网站状态并捕获视觉证据
- **报告生成**：生成 Web 仪表板的 PDF 或截图
- **多步骤流程**：使用会话持久性将浏览器任务链接在一起

## 故障排除

如果你遇到 Browser Use 节点的问题：

- 验证你的 API Key 有效且有足够的积分
- 确保你的指令清晰具体
- 对于复杂任务，考虑将其分解为多个步骤
- 参考 [Browser Use 文档](https://docs.browser-use.com) 获取指令最佳实践
