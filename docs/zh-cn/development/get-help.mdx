## 社区支持

Browser Use 拥有活跃的开发者社区，如果你遇到问题或需要帮助，可以通过以下渠道获得支持：

### 1. GitHub Issues

查看 [GitHub Issues](https://github.com/browser-use/browser-use/issues) 获取：
- 已知问题和解决方案
- 功能请求和讨论
- 错误报告和修复

### 2. Discord 社区

加入我们的 [Discord 社区](https://link.browser-use.com/discord) 获取：
- 实时问答和讨论
- 社区支持和建议
- 最新更新和公告

### 3. 企业支持

对于企业级支持需求，请联系：
- 邮箱：support@browser-use.com

我们提供：
- 优先技术支持
- 定制解决方案
- 专业咨询服务

## 常见问题

### 如何报告 Bug？

1. 搜索现有 Issues 确认是否已报告
2. 创建新的 Issue
3. 描述问题、复现步骤和预期行为
4. 提供日志、截图等辅助信息

### 如何请求新功能？

1. 在 GitHub Issues 中搜索类似请求
2. 创建新 Issue 并使用「Feature Request」标签
3. 描述功能用途、使用场景和实现建议

### 如何贡献代码？

1. Fork 本仓库
2. 创建功能分支
3. 提交更改
4. 创建 Pull Request
